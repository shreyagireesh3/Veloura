🌸 Veloura
" See what users never say "

Veloura is a UX intelligence platform for e-commerce that detects hidden user frustration in real time using anonymous behavior signals like repeated clicks, hesitation, and abandoned actions.

🚀KEY FEATURES
🧠 Silent behavior tracking
⚡ Live hesitation detection
💬 Smart messaging support
📊 Analytics dashboard
🕶️ Spatial Map (UX hotspots)
🔒 Privacy-friendly design

TECH STACK
⚛️ React
📘 TypeScript
🎨 CSS / Tailwind
🐙 GitHub
▲ Vercel
